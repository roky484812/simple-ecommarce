# **Full-Stack E-Commerce Application Development Task**

## **Objective**

Develop a modern, responsive, and fully functional **Simple Full-Stack E-Commerce Application** that demonstrates best practices in software architecture, database design, user experience, and deployment.

## **Scope of Work**

The project should include the following components:

### **1\. Front-End Development**

- Responsive and modern user interface
- Intuitive navigation and user experience
- Optimized performance across desktop and mobile devices

### **2\. Database Design**

- Well-structured and normalized database schema
- Efficient relationships and data integrity
- Scalable design to support future enhancements

### **3\. Back-End Development**

- Secure and maintainable server-side architecture
- RESTful APIs or equivalent backend services
- Proper authentication, validation, and business logic implementation

## **Required Views**

The application should include the following interfaces:

- **Storefront** - Customer-facing e-commerce website
- **Admin Panel** - Administrative dashboard for managing the platform
- **Modern UI/UX** - Clean, responsive, and user-friendly design across all views

## **Required Features**

The application must support the following core functionalities:

- Product & Category Management
- Customer Management
- Order Management
- Shopping Cart
- Payment History
- Customer Order Tracking

## **Deployment Requirement**

The completed application **must be deployed** by the candidate and accessible through a live URL. Candidates should ensure the deployed application is fully functional and available for demonstration.

## **Onsite Interview Assessment**

Candidates who are shortlisted for the onsite interview will be required to:

- Demonstrate the deployed application and explain its architecture, implementation, and key features.
- Implement a new feature within a specified timeframe using **AI agentic prompting techniques**, demonstrating effective AI-assisted software development, problem-solving, and integration skills.
